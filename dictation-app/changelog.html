Dictation App

Whisper-powered system-wide dictation for Windows — release changelog

Release History

Audio Feedback Feature March 28, 2026
Added
  • Start tone — two ascending sine tones (600 Hz → 900 Hz) play immediately after the microphone opens, confirming the app is listening
  • Stop tone — two descending sine tones (700 Hz → 440 Hz) play when recording stops, confirming audio capture has ended
  • Tones are silent on discard — pressing Discard produces no audio feedback so accidental triggers stay unobtrusive
  • Each tone uses a self-contained AudioContext that opens and closes automatically; no persistent audio resources are held between recordings
Tray Icons Polish March 7, 2026
Changed
  • mic-on.png / mic-off.png replaced with properly sized assets generated by generate-icons.js
  • Icons now render crisply at Windows tray scale (16 px and 32 px DPI variants)
v1.0.0 — Initial Release New March 7, 2026
Core Dictation
  • Global hotkey (Ctrl+Alt+D) toggles recording system-wide without stealing window focus
  • Whisper transcription — audio is captured via MediaRecorder and sent to OpenAI Whisper (whisper-1) for transcription; supports language selection
  • Auto-paste — transcribed text is written to the clipboard and sent as Ctrl+V to the foreground window via PowerShell WScript; target window never loses focus
  • Silence detection — recording auto-stops after configurable silence (default 2 500 ms) using an RMS analyser on the live audio stream
  • Spoken newline commands — phrases “new line”, “newline”, “next line”, and “new paragraph” are converted to actual newline characters before pasting
Claude Cleanup (Optional)
  • Raw Whisper transcript can be passed through a Claude model to remove filler words, fix punctuation, and join run-on sentences
  • Configurable model (defaults to claude-haiku-4-5); falls back to raw transcript on API error
  • Indicator shows a distinct “Cleaning up…” state while Claude processes
Floating Indicator
  • Non-focusable pill appears at the bottom-center of the screen during recording — never steals focus from the target window
  • Six visual states: Recording, Paused, Processing (spinner), Cleaning, Success (shows transcript preview), Error
  • Active microphone label displayed in the pill
  • In-pill controls: Pause / Resume, Stop (commit recording), Discard (discard audio without transcribing)
  • Card is draggable to reposition on screen
System Tray
  • Tray icon switches between mic-off and mic-on states; tooltip reflects current status (Ready / Recording / Transcribing)
  • Left-click tray icon to toggle recording; right-click for context menu
  • Context menu: Start/Stop Dictation (with active hotkey shown), Settings, Quit
Settings
  • OpenAI API key and Anthropic API key configuration
  • Claude cleanup toggle and model selector
  • Hotkey rebinding with live re-registration (fallbacks: Ctrl+Alt+D → Ctrl+Shift+D)
  • Language selection for Whisper
  • Auto-paste toggle
  • Silence timeout (ms) — set to 0 to disable auto-stop
  • Settings open automatically on first launch if no API key is configured
