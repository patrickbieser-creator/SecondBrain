# AGENTS.md (repo root)

## Goal
Build "SecondBrain" as a personal project manager / second-brain web app.

## How to work in this repo
- Read Plans.md first. Keep work aligned to it.
- Prefer small, reviewable commits. Summarize changes in each commit message.
- If you need to make a big decision, propose options in the PR description before implementing.
- Add/update docs when behavior changes.
- Do not touch any sibling folders outside this repo.

## Project files
- Plans.md = product spec + roadmap + priorities
- /docs = architecture + decisions (ADRs)
- /src = application code
